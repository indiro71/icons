# Icons library

## Installation

```bash
npm i @indiro/react-icons-library
```

## Usage

```jsx
import { ArrowRight } from '@indiro/react-icons-library';

<ArrowRight style={{width: 50}} title="Right arrow" />
```
## Add new Icons

 - copy SVG icon in `assets`
 - run command

```bash
npm run release
```
